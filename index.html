나만의 웹사이트입니다.

이 페이지는 트렌디한 다크 모드 글래스모피즘 디자인으로 제작되었습니다. GitHub Pages를 통해 누구나 쉽게 웹사이트를 가질 수 있습니다.

환영합니다 ✨

방문해주셔서 감사합니다. 하나의 HTML 파일 안에 스타일(CSS)과 내용이 모두 포함되어 있어 매우 가볍고 빠릅니다.

당신의 포트폴리오, 프로젝트 소개, 혹은 간단한 자기소개서로 언제든지 수정해서 사용할 수 있습니다.

GitHub Pages 사용법 🚀

  • 1 GitHub에 로그인 후 username.github.io 이름으로 새 Repository를 생성합니다.
  • 2 생성한 레포지토리에 이 index.html 파일을 업로드/푸시합니다.
  • 3 레포지토리의 Settings > Pages 메뉴로 들어갑니다.
  • 4 Source를 main 브랜치(또는 master)로 설정하고 저장합니다.
  • 5 잠시 후 https://username.github.io 로 접속하면 페이지 완성!
